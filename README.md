## Learning Approach

### DSA Strategy

I follow a structured, pattern-based approach:

1. Learn the concept and underlying time/space complexity.
2. Solve 3–5 basic problems.
3. Progress to medium and hard variants.
4. Revisit the topic after 1–2 weeks for retention.
5. Document mistakes, optimizations, and alternate approaches.
